#Chatty App
=====================

A one page chatroom that uses websockets and react.

![alt text](https://github.com/Matt-Greff/react-simple-boilerplate/blob/master/docs/demo-image.png?raw=true)

### Setup

Install the dependencies and start the server.

```
npm install
npm start
cd chatty_server && npm install && npm start
open http://localhost:3000
```

### Chatty Server Dependencies
* uuid
* express
* ws

### Dependencies

* babel-core
* babel-loader
* babel-preset-es2015
* babel-preset-react
* node-sass
* sass-loader
* sockjs-client
* style-loader
* webpack
* react
* react-dom

### Dev Dependencies

* webpack-dev-server
* css-loader
